¿Qué es un editor de texto web?

¿Qué es un editor de texto web? Es un aplicativo creado con el fin de facilitar la creación o construcción de páginas web o contenido web a diferencia de un editor común que permite editar el texto sin formato alguno, el editor web el cual permite editar datos y guardarlos en un formato web conocido de tal manera que puede mostrar en colores resaltados palabras clave de un lenguaje de programación u interpretador de texto; adicionalmente se debe dar claro que un editor de texto web es diferente al procesador de texto ya que este permite crear documentos más elaborados con formatos específicos y con diseños orientados a ser impresos.

Los editores web cuentan además con una serie de funciones especializadas para la creación de sitios web que permiten una mayor practicidad del desarrollo de un proyecto por parte de un diseñador o desarrollador.
